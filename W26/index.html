PICT B.E. Course Structure

BE Computer Engineering

First Year (FE)

  • Engineering Mathematics I: Calculus and differential equations.
  • Physics: Mechanics and optics fundamentals.
  • Chemistry: Material science and chemical properties.
  • Basic Electrical Engineering: Circuits and electrical systems.
  • Programming in C: Introduction to programming.

Second Year (SE)

  • Data Structures: Arrays, lists, and trees.
  • Discrete Mathematics: Logic and set theory.
  • Digital Electronics: Logic gates and circuits.
  • Object-Oriented Programming: Java and C++ concepts.
  • Computer Organization: CPU and memory architecture.

Third Year (TE)

  • Database Management Systems: SQL and database design.
  • Operating Systems: Process and memory management.
  • Software Engineering: SDLC and agile methodologies.
  • Computer Networks: TCP/IP and networking protocols.
  • Microprocessors: Architecture and interfacing.

Final Year (BE)

  • Machine Learning: Algorithms and data modeling.
  • Cloud Computing: AWS and virtualization.
  • Information Security: Cryptography and cybersecurity.
  • Project Work: Capstone project implementation.
  • Elective (e.g., AI): Advanced topics in AI.
